O que é Git?
Desenvolvido por Linus Torvalds para o controle de versão desenvolvimeno do kernel linux, hoje utilizado por varias empresas para controle de versão.
Diferente de outros softwares de controle de versão que guarda as alterações realizados em um determinado arquivo o Git tira snapshots de todos os arquivos do projeto e não só do arquivo que foi commitado. (commitar é enviar alterações para o controle de versão).

ÁREAS DE OPERAÇÃO
Git Directory: é onde o Git guarda os dados e objetos do seu projeto. Ele é o diretório mais importante do Git e é ele que será copiado quando alguém clonar (clonar é copiar o projeto para a sua máquina) o projeto.
Work Directory: é onde você vai trabalhar. Os arquivos ficam aí para poderem ser usados e alterados quantas vezes quiser para você. É basicamente sua pasta de arquivos dos projeto.

INSTALANDO O Git NO WINDOWS
Download do Git: http://msysgit.github.io/
Ele vai instalar para você os comandos do Git para serem usados no terminal e uma uma interface padrão para quem não está acostumado a usar linhas de comando. Após instalar o Git é necessário gerar uma chave ssh e fazer o cadastro em algum repositorio Git é necessário apontar a chave publica no GitHub.Comandos para realizar a alteração
1º Git status 2º Git add arquivo 3º git commit -m "" 4º Git push
