Diferença entre linguagem compilada e interpretada

Linguagem Compilada: Todo o texto do programa é traduzido (compilado) de maneira que as instruções possam ser realizadas pelo computador. A linguagem compilada geralmente gera um executável (como os famosos ".exe") .Exemplos: Pascal, C.
Linguagem interpretada: O texto do programa é traduzido à medida que vai sendo executado. Exemplos: Javascript, Perl, Ruby.
