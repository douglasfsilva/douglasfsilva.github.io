BootStrap

O Bootstrap é o framework criado por Mark Otto e Jacob Thornton, ambos engenheiros do Twitter e segundo a Wikipedia, é uma coleção de ferramentas para criação de websites e aplicações web utilizando o HTML e CSS.
O objetivo de um framework é facilitar o desenvolvimento/manutenção de um projeto utilizando reaproveitamento de código (bibliotecas). Muitos dos frameworks utilizam o padrão de arquitetura MVC (Model/View/Controller). O CMS Joomla é um deles !
Voltando ao Bootstrap, pode-se dizer que ele oferta uma coleção de scripts CSS e jQuery (biblioteca javascript) com intuito de dinamizar as interfaces e componentes de uma página web. Alguns plugins desempenham funções, como por exemplo, permitir o auto-preecnhimento de um campo input. Outros permitem funções como Modal, Dropdown, Scrollspy, Tab, Tooltip, Popover, Alert, Button,Collapse, Carousel e Typeahead. Não podemos nos esquecer que a partir da versão 2 ele suporta o design responsivo

layoutit

Layoutit! é um construtor de interfaces que utiliza o Bootstrap para definir os estilos e elementos a serem utilizados. O seu funcionamento é muito simples, basta escolher o tipo de recurso que deseja, clicar, arrastar e soltar onde quiser.
Você poderá definir o grid da página formatando todo o conteúdo como achar melhor. É possível criar páginas com várias colunas de conteúdo e se o resultado final não for o desejado basta voltar, excluir e refazer o que for necessário.
Depois de finalizado é só fazer o download dos arquivos. Se quiser apenas o HTML é só copiar e utilizar em qualquer projeto. Download e Informações
O Layoutit! pode ser utilizado por qualquer pessoa. O aplicativo ainda esta na versão Beta, mas tudo indica que em breve ele deverá ter planos de acesso, por isso não perca tempo e comece a utilizar o quanto antes.
