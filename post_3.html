Nesta aula, após recordamos sobre os itens aprendidos na última aula, como: tags de criação de documento html, tags básicas, estilos etc, apredemos de maneira correta a chamada de um arquivo de estilos, chamado estilos.css, que substitui todo o código que estava dentro da própria index por um arquivo externo, visando assim diminuir os códigos de cada página e facilitar na formatação das mesmas, visto que se aplica a qualquer página que contenham as tags que foram formatadas no arquivo css.
